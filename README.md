## Задание 2.

В задании 2 добавить к реализованному в первом задании классу указанные в варианте
перегруженные операции.
В главной функции main показать работу всех функций с дружественным интерфейсом.
Необходимо решить задания согласно вашему варианту.



### Название класса 
Time <br>
### Унарные операции:
++ добавление минуты к объекту типа Time <br>
-- вычитание минуты из объекта типа Time<br>
### Операции приведения типа:
short int (явная) – результатом является количество часов (минуты
отбрасываются);<br>
bool (неявная) – результатом является true, если часы и минуты не равны
нулю, и falseв противном случае.<br>
### Бинарные операции:
plus Time t, беззнаковое целое число (минуты). Результат должен быть типа
Time.<br>
– Time t, беззнаковое целое число (минуты). Результат должен быть типа
Time.<br>

## Алгоритм решения:

### plus Time t
Перегружаю оператор +. Создаю две переменные unsigned int totalMinutes и unsigned int totalHours. В общие минуты записываем сумму текущих минут, а в общих часах прибаляем минуты деленные нацело на 60. Затем берем остаток от деления на 60 всех минут и записываем его в общие минуты. Возвращаем объект типа Time с новым временем.

### - Time t
Перегружаю оператор -. Создаю две переменные long long totalCurrentMinutes и long long resultMinutes. Умножааю часы на 60, тем самым получая и минуты и привожу их к типу long long. Результат записываю в переменную totalCurrentMinutes. Дальше из этих общих минут вычитается количество минут, которое было на входе функции. Результат записывается в  resultMinutes. Если ответ меньше нуля, то возвращается объект с нулевым временем. Если больше нуля, то часы и минуты приводятся к нормальномму виду и возвращется объект с новым временем.

### ++ добавление минуты к объекту типа Time
Данный оператор может быть как посификсный, так и постфиксный. Сначала разберу постфиксный. В переменную temp записывает текщее значение объекта. Вычитаем из объекта 1 минуту. Возвращаем исходный объект. 
В префиксном сначал вычитаем 1 минуту, затем если минут больше или равно 60, то вычитаем 60 минут и добавляем 1 к часам, чтобы нормализовать время. Возвращаем объект.

### -- вычитание минуты из объекта типа Time
Данный оператор может быть как посификсный, так и постфиксный. Сначала разберу постфиксный. В переменную temp записывает текщее значение объекта. Прибавляем к объекту 1 минут. Возвращаем исходный объект. 
В префиксном если минут больше 0, то вычитаем 1 минуту. Если часов больше 0, то вычитаем 1 час и значение минут ставил 59. Возвращаем объект.

### short int (явная)
В данном операторе будет прописано, как объект Time будет приводиту к типу short int. Возвращаем static_cast<short int>(hours) - количество часов с типом short int.

### bool (неявная)
В данном операторе будет неявно преобразовываться объект Time к типу bool. Возвраем true если часы или минуты не равно 0, и false если оба раные нулю

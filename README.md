### Название класса 
Time <br>
### Поля 
byte hours, byte minutes <br>
### Метод 
Добавление произвольного количества
минут (unsigned int) к объекту типа Time.
Результат должен быть типа Time.

## Алгоритм решения:

В классе реализивано два поля hours и minutes с типом unsigned int.
Прописываю три конструктора: перывый стандартный, который ничего не принимает и создает объект с нулевым временем, второй создает объект только с количеством часов и с нулевыми минутами, третий создает объект с часами и минутами, причем приводит ввод в нормальный вид так, чтобы количество минут не превышало 60.
Создаю метод add_minutes(unsigned int min) который добавляет минуты к объекту. В нем берется ссылка на объект и к нему прибавляются минуты.
